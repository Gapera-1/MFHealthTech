## Deployment to Vercel

This website is ready to be deployed to Vercel. Here are the steps:

### Method 1: Deploy Directly from GitHub (Recommended)

1. Push this code to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click "New Project" and import your repository
4. Vercel will automatically detect this is a static site and deploy it

### Method 2: Using Vercel CLI

1. Install the Vercel CLI globally:
   ```bash
   npm install -g vercel
   ```

2. Run the deploy command from your project directory:
   ```bash
   vercel
   ```

3. Follow the prompts to link to your Vercel account and configure the project

4. To deploy to production:
   ```bash
   vercel --prod
   ```

### Local Development

To run this site locally for development:

```bash
npx serve .
```

Or if you have the Vercel CLI installed:
```bash
vercel dev
```
